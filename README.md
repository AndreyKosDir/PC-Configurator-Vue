# PC Configurator VUE

![Demonstration](demonstration.gif)

## Description
```
Разработан конфигуратор компьютера, который позволяет выбрать
отдельные детали ПК в каталоге и рассчитать общую стоимость
на основании выбранных деталей.
```

npm install
